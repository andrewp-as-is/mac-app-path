### Installation
```bash
$ [sudo] pip install mac-app-path
```

```bash
$ [sudo] npm i -g mac-app-path
```

#### Examples
```bash
$ app-path "Google Chrome"
/Users/username/Applications/Google Chrome.app
$ app-path "Activity Monitor"
/Applications/Utilities/Activity Monitor.app
```
